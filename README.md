# NTHU Student Phone Addiction Analysis

## Project Overview
This project investigates the mobile phone usage habits and addiction patterns among students at National Tsing Hua University (NTHU). 

Using statistical analysis and data visualization, the project aims to identify the correlation between usage time, usage purpose (e.g., social media, gaming), and the "Mobile Phone Addiction Index" (MPAI).

## Technologies Used
* **Language:** R
* **Tools:** RMarkdown
* **Libraries:** `ggplot2`, `dplyr` (Data manipulation and Visualization)

## File Description
* `Phone_Addiction_Analysis.Rmd`: The source code written in RMarkdown, containing data cleaning, statistical testing, and plotting logic.
* `Analysis_Report.html`: The generated full report. Download this file and open it in any web browser to view the results and charts.

## Key Analysis
1.  **Exploratory Data Analysis (EDA):** Visualizing the distribution of phone usage hours among different student groups.
2.  **Statistical Testing:** Analyzing the relationship between sleep quality and phone usage.
3.  **Addiction Scoring:** Calculating addiction levels based on survey responses.
